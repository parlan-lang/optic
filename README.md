# The Optic Compilation Backend

A fast, simple but powerfull compilation backend that aims to make compiler development easier

## The Vision

#### the vision is to make a backend:  
- simple enough to be learned in a weekend
- fast enough to do not affect the development
- and powerfull enough to be used in real aplications

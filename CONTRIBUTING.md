# Contributing to Optic

First off, thank you for considering contributing to Optic! We truly appreciate all kind of contributions, it doesn't matter if you are adding a new optimization pass or just fixing a typo in the docs!

To contribute to Optic, follow these steps:

1. **Fork the repository** and create your branch from `main`
2. **Make your changes** (and keep your code clean)
3. **Open a Pull Request** with a clear title and a description of what you changed

# Code of Conduct

Our code of conduct is simple: **Be kind, with everyone**

Optic is a welcoming project for everyone, regardless of experience, just like Parlan. Be respectful, helpful, and patient with anyone

Thank you for helping make Optic better!
